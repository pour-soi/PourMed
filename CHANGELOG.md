# Changelog

## 1.0.0 - 2026-07-21

- Released the installable PourMed PWA with an offline application shell.
- Added configurable medication schedules, reminders, snoozing, and completion tracking.
- Added Web Push notifications and iPhone Home Screen support.
- Added secure bearer-token authentication with a secret verifier.
- Added medication history, notes, corrections, exports, statistics, and streaks.
- Added the Cloudflare Worker, SQLite Durable Object, service worker, automated tests, and self-hosting documentation.
